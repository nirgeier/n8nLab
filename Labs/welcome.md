# Welcome to the n8n Deep Dive

---

This intensive hands-on training will take you from n8n basics to production deployment.

## What You'll Learn

- **Installation & Architecture**: Install n8n, Docker Compose setup
- **Triggers & Data Flow**: Trigger types, expressions, webhooks
- **Database Operations**: PostgreSQL/Supabase CRUD, sync, bulk ops
- **HTTP & APIs**: HTTP Request node, authentication, API chaining
- **Email & Notifications**: Templated emails, notification hubs
- **File Processing**: CSV, PDF, OCR, batch processing
- **AI Integration**: Claude/OpenAI in workflows, classification
- **Error Handling**: Retry logic, dead letter queues, monitoring
- **Advanced Patterns**: Sub-workflows, looping, conditional routing
- **Production Deployment**: Security, backups, monitoring, runbooks

## Prerequisites

- A computer with terminal access
- Docker installed
- n8n Cloud account (or self-hosted n8n)
- Basic understanding of APIs and webhooks
